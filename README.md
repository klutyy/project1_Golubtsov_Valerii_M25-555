# Labyrinth Game

Добро пожаловать в **Labyrinth Game** — текстовое приключение, где игрок исследует комнаты, решает загадки, избегает ловушек и в конце находит сокровище.  
Проект написан на Python и использует простую логику обработки команд, событий и случайных ситуаций.

---

## Установка

### Вариант 1 — через Poetry
```bash
poetry install
```

### Вариант 2 - через Makefile

```
make install
```

## Запуск игры

### Вариант 1 — через Poetry
```bash
poetry run project
```

### Вариант 2 - через Makefile
```
make project
```
# Список команд для игры

| Команда          | Описание                                                 |
| ---------------- | -------------------------------------------------------- |
| `go <direction>` | Перейти в направлении (`north`, `south`, `east`, `west`) |
| `look`           | Осмотреть текущую комнату                                |
| `take <item>`    | Поднять предмет                                          |
| `use <item>`     | Использовать предмет из инвентаря                        |
| `inventory`      | Показать содержимое инвентаря                            |
| `solve`          | Решить загадку или попытаться открыть сундук             |
| `help`           | Показать список команд                                   |
| `quit` / `exit`  | Выйти из игры                                            |
